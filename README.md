### Keaton Taliaferro

**I build verifiable AI tools for nonprofits.** Tools whose claims a human can check in seconds, instead of trusting a model's say so.

The easy version of "AI for nonprofits" is a thin wrapper around an agreeable model. In a setting where a wrong "yes" costs an organization a grant or ships a broken laptop to a family, I build the harder, more useful version: the model observes, and deterministic code you can audit makes the decision.
